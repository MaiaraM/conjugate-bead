# Conjugate Bead

Package to return a collection of Conjugates of integers from another collection of integers (Transposed Matrix). was use in this project **ESLint**, **Typescript** and **Jest** were used for the unit tests

```text
 npm install
 ```

## Start Application

```text
 npm run start 
 ```

## Test

Run test: 

```text
 npm run test 
 ```
